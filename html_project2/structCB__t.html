Detailed Description

This typedef defines a data type CB_t, to represent a struct corresponding to the circular buffer. The elements of the struct are: buf – pointer to uint8_t data contained within the buffer head – uint8_t pointer to the 'write' location tail – uint8_t pointer to the 'read' location size – uint16_t type size of circular buffer count – uint16_t type number of data elements in the buffer
